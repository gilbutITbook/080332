## 실습 환경
이 책의 실습을 검증한 환경은 둘로 나눌 수 있다.
</br>

#### (1) 시스템 독립적인 예제의 실행 환경: Go Playground
- https://go.dev/play
- 예: 슬라이스 연산 등 코어 기능
- 고 언어: 1.20

#### (2) 시스템 종속적인 예제의 실행 환경: Go v1.20.5 / 맥OS(M1)
- 프로세서: M1(ARM 기반, darwin/arm64)
- 운영체제: macOS 12.5 Monterey
- 예: I/O, 고루틴 성능 비교 등
- 고 언어: 1.20.5
